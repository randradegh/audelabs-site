# Guía de Contribución

## Desarrollo

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Crear una rama para tu feature: `git checkout -b feature/nombre-feature`
4. Hacer cambios y commits descriptivos
5. Push a la rama: `git push origin feature/nombre-feature`
6. Crear un Pull Request

## Estándares de Código

- Usar TypeScript para todos los componentes
- Seguir las convenciones de Next.js 14 (App Router)
- Usar Tailwind CSS para estilos
- Mantener componentes pequeños y reutilizables
- Agregar comentarios cuando sea necesario

## Commits

Usar mensajes de commit descriptivos en español:

```
feat: agregar nueva sección de servicios
fix: corregir navegación en móvil
docs: actualizar README
style: mejorar diseño del hero
refactor: simplificar componente de formulario
```
