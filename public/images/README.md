# Directorio de Imágenes

Este directorio contiene todas las imágenes estáticas del sitio web.

## Estructura

```
public/images/
├── audelabs-terra/          # Imágenes para la página de Audelabs Terra
│   ├── hero-platform.jpg
│   ├── gestion-centralizada.jpg
│   ├── presentacion-desarrollos.jpg
│   └── captura-prospectos.jpg
└── README.md                # Este archivo
```

## Uso en el código

Las imágenes en la carpeta `public` son accesibles directamente desde la raíz del sitio. Por ejemplo:

- Archivo: `public/images/audelabs-terra/hero-platform.jpg`
- URL en código: `/images/audelabs-terra/hero-platform.jpg`

## Convenciones

1. **Nombres de archivos**: Usar minúsculas y guiones (kebab-case)
2. **Formatos**: Preferir WebP o JPG optimizados
3. **Tamaños**: Optimizar imágenes antes de subirlas
4. **Organización**: Agrupar por sección/página

## Optimización

Antes de subir imágenes:
- Comprimir usando herramientas como TinyPNG o Squoosh
- Usar formatos modernos (WebP) cuando sea posible
- Mantener tamaños razonables (< 500KB por imagen)
