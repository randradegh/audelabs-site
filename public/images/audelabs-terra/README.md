# Imágenes de Audelabs Terra

Coloca aquí las imágenes definitivas para la página de Audelabs Terra.

## Estructura de archivos

```
public/images/audelabs-terra/
├── hero-platform.jpg          # Imagen principal del hero (800x600 o mayor)
├── gestion-centralizada.jpg   # Imagen de gestión de propiedades (800x600 o mayor)
├── presentacion-desarrollos.jpg # Imagen de presentación profesional (800x600 o mayor)
└── captura-prospectos.jpg     # Imagen de captura de prospectos (800x600 o mayor)
```

## Especificaciones recomendadas

- **Formato**: JPG o WebP (WebP recomendado para mejor compresión)
- **Dimensiones mínimas**: 800x600px
- **Dimensiones recomendadas**: 1200x900px o 1600x1200px
- **Peso máximo**: 500KB por imagen (optimizar antes de subir)
- **Aspecto**: Mantener proporción 4:3 o 16:9

## Nombres de archivos

Los nombres deben coincidir exactamente con los especificados arriba para que el código los encuentre automáticamente.

## Optimización

Antes de subir las imágenes, optimízalas usando herramientas como:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- [ImageOptim](https://imageoptim.com/)
